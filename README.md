# Player EV Decision Engine

A decision-support tool for evaluating casino offer value, tier progression, and expected value (EV).

## 🚀 Overview

This app helps players determine whether continuing to play toward the next tier is worth it based on expected value modeling.

## 📊 Key Features

* Expected value (EV) calculation
* Tier progression tracking
* Offer breakdown (freeplay, comps, etc.)
* Decision recommendation engine

## 🛠 Tech Stack

* Python
* Streamlit
* Pandas

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## ⚠️ Disclaimer

This tool is for educational and analytical purposes only.
It does not represent or reverse-engineer any real casino systems or proprietary algorithms.
All values are hypothetical and based on user-reported data and modeling assumptions.
